## About Laravel Glowbabe Shop API

**Glowbabe Shop API** is builded for an api service backend for Flutter Mobile App in final project Year 4 in Royal University of Phnom Penh. It was hosted as a web using [Railway](https://railway.com/).

## How to run:

1. Clone the project

```
https://github.com/namponleu/laravel-skincare-api.git
```

2. Using composer to install some missing configuration file

```
composer install
```

3. Run migrate

```
php artisan migrate
```

4. Run on your local

```
php artisan serve
```

**Note:** make sure you have create a mysql database already and connected.
But if not we have created a schema to create a file automatic.
